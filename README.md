mod2mp3
=======

A tiny shell script to convert MOD files to MP3, using FFMPEG.

Usage
=====

```
$ ./mod2mp3 
Error: please specify a *.mod filename as argument
Usage: ./mod2mp3 file.mod
```

Dependencies
============

You need to have the ffmpeg available in your PATH.
